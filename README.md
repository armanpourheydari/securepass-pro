# 🔐 SecurePass Pro

A Python tool to **generate strong passwords** and **check password strength** with detailed suggestions.

## Features

- ✅ **Password Generator** – Create strong, random passwords
- ✅ **Password Checker** – Check strength and get improvement suggestions
- ✅ **Breach Detection** – Detect common and leaked passwords
- ✅ **Pattern Detection** – Detect sequential numbers, keyboard patterns, and repeated characters
- ✅ **Score System** – 0-100 scoring with detailed feedback
